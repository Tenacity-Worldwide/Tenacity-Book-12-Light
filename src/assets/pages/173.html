What is the difference between the verb forms used in questions 2 and 5? Why is each of
them being used in these questions?
We use the present perfect continuous to talk about things that have been happening and are
still going on.
For example:
• School has been going quite well.
• I’ve been working quite steadily all year.
It emphasises that something is still going on.
It is a bit different from the structure which is used to talk about things that have happened –
that have been completed at some point before the present time: the present perfect simple.
For example:
• I’ ve learned a number of skills working in my uncle’s shop.
• I’m quite pleased with what I’ve achieved this year.
